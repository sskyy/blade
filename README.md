Blade  V0.0.1
==============

**This plugin currently only work for Sketch beta 3.0.2(7779). This version is unstable now, and may crash due to Sketch [memory issue](https://github.com/tisho/sketch-plugins/issues/2#issuecomment-44082938). You may need at least 3GB free memory to generate HTML for complicate sketch resource file.**

##What is blade?##

Blade is a Sketch 3 plugin for automatically HTML generating. It will generate tag `<div>` for group, tag `<p>` for text , etc.
