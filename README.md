##What is blade?##

Blade is a Sketch 3 plugin for automatically HTML generating. It will generate tag `<div>` for group, tag `<p>` for text , etc. 

##How to Install?##

Just place `blade.sketchplugin` and `blade` folder in your Sketch plugin folder.
